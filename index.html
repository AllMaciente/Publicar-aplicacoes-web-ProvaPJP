3.Explique o que é um servidor compartilhado e um servidor dedicado, evidenciando suas vantagens e desvantagens.

Compartilhado: Um servidor usado por varias pessoas

Vantagens: Ele e mais Barato

Desvantagens: Se um site sobrecarregar ele vai cair todos os outros que estão no mesmo servidor

Dedicado: Um servidor usado somente para aquela aplicação

Vantagens: vai ser mais forte

Desvantagens: tem um preço maior

5.O que é um servidor de banco de dados, explique como fazer o backup e restaurar  esse backup.

Um servidor que vai hospedar um banco de dados para ser acessado por outras aplicação via web

Backup

para fazer um backup do banco de dados podemos usar esse comando

mysqldump -u usuario -p banco_de_dados > backup.sql

para compactar o backup podemos usar esse comando

zip backup_banco.zip backup.sql

Restaura Backup

Descompactar Backup

unzip backup_banco.zip

Importar para o banco de dados

mysql -u usuario -p banco_de_dados < backup.sql

6.O que é um serviço ssh, e como ele permite acessar um servidor web para transferir arquivos de uma máquina local e o servidor.

SSH: E um protocolo usado para fazer login em servidores remotos de uma forma segura

para a transferência de aquivos podemos usar o scp que usa o ssh para transferir entre maquina local e o servidor

Estrutura básica:

scp [opções] [usuário_origem@IP]:/[caminho/arquivo] [usuário_destino@IP]:/[diretório_destino]

Componentes:
  • [opções]: Modificadores extras
  • [usuário_origem@IP]: Usuário e IP de origem
  • :*/ : Indica início do caminho
  • [caminho/arquivo]: Local e nome do arquivo
  • [usuário_destino@IP]: Usuário e IP de destino
  • [diretório_destino]: Local onde o arquivo será salvo
Exemplo prático:

scp root@123.123.123.12:/users/Senac/Desktop/SCP.png root@456.456.456.45:/home/destino/

8.Explique alguns comando básicos do Linux.

  • ls — Lista arquivos e pastas no diretório.
  • cd <diretório> — Muda para o diretório especificado.
  • mkdir <nome> — Cria um novo diretório.
  • rm -rf <nome> — Remove diretórios/aquivos
  • touch <arquivo> — Cria um arquivo vazio.
  • cp <origem> <destino> — Copia arquivos ou diretórios.
  • mv <origem> <destino> — Move ou renomeia arquivos/diretórios.
  • rm <arquivo> — Remove arquivos.
  • cat <arquivo> — Mostra o conteúdo do arquivo.
  • nano <arquivo> — Editor de texto em linha de comando.
  • sudo <comando> — Executa um comando como superusuário.
  • apt update — Atualiza a lista de pacotes (Debian/Ubuntu).
